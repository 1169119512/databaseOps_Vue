# 1.项目介绍

vue是一个基于 Vue 3 + Vite + Element Plus 的数据库管理前端项目，支持动态管理多个后端数据库连接，可对数据库表进行增删查改的操作，同时支持条件筛选和批量删除功能。

# 2.运行方法

## 2.1.安装依赖

```bash
npm install
```

## 2.2.启动开发服务器

```bash
npm run dev
```

浏览器访问：`http://localhost:80`

## 2.3.配置数据库连接

1. 进入「连接管理」页面
2. 点击「添加连接」
3. 填写以下信息：

```
连接名称：供应商数据库
后端地址：http://localhost:5000
模块名：supplier-archive
描述：供应商档案管理系统
```

或者导入配置文件进行批量添加，实例配置文件可见database_connections_template.json

4. 点击「测试」验证连接
5. 点击「激活」激活连接

## 2.4开始使用

进入「数据管理」页面，即可进行增删改查操作！

## 2.5.构建部署

```bash
npm run build
```
