# 快速开始指南

## 🚀 5分钟快速启动

### 第一步：安装依赖

```bash
npm install
```

### 第二步：启动开发服务器

```bash
npm run dev
```

浏览器访问：`http://localhost:80`

### 第三步：配置数据库连接

1. 进入「连接管理」页面
2. 点击「添加连接」
3. 填写以下信息：

```
连接名称：供应商数据库
后端地址：http://localhost:5000
表名：supplier-archive
描述：供应商档案管理系统
```

4. 点击「测试」验证连接
5. 点击「激活」激活连接

### 第四步：开始使用

进入「数据管理」页面，即可进行增删改查操作！

## 📝 配置说明

### 开发环境配置

创建 `.env` 文件（如果不存在）：

```bash
# 复制示例配置
cp .env.example .env
```

修改配置：
```env
VITE_API_BASE_URL=http://localhost:5000
```

### 生产环境配置（sakurafrp）

1. **安装 sakurafrp 客户端**
   - 访问：https://www.natfrp.com/
   - 注册账号并下载客户端

2. **创建隧道**
   - 协议：HTTP
   - 本地地址：127.0.0.1
   - 本地端口：5000（后端服务端口）
   - 获得外网地址（例如：`https://abc123.frp.sakura.com`）

3. **修改生产环境配置**

编辑 `.env.production`：
```env
VITE_API_BASE_URL=https://abc123.frp.sakura.com
```

4. **构建部署**
```bash
npm run build
```

## 🎯 功能演示

### 数值筛选示例

筛选 ZBXH（主键）大于等于10且小于等于100的记录：

1. 展开「高级筛选」
2. 在 ZBXH 字段：
   - 选择运算符 ">="
   - 输入值 "10"
3. 添加另一个 ZBXH 筛选：
   - 选择运算符 "<="
   - 输入值 "100"
4. 点击「查询」

### 模糊查询示例

查找公司名称包含"科技"的记录：

1. 展开「高级筛选」
2. 在 GYSMC（供应商名称）字段：
   - 输入 "科技"
3. 点击「查询」

### 日期范围查询示例

查询2020年到2024年成立的公司：

1. 展开「高级筛选」
2. 在 CLSJ（成立时间）字段：
   - 选择开始日期：2020-01-01
   - 选择结束日期：2024-12-31
3. 点击「查询」

## 🔧 常见问题

### Q1: 启动时端口被占用怎么办？

修改 `vite.config.js` 中的端口配置：

```javascript
server: {
  port: 8080, // 改为其他端口
  // ...
}
```

### Q2: 连接测试失败怎么办？

检查以下几点：
1. 后端服务是否已启动
2. 后端地址是否正确
3. 表名是否存在
4. 网络是否畅通
5. CORS 是否正确配置

### Q3: 如何切换不同的数据库？

1. 进入「连接管理」
2. 添加多个连接配置
3. 点击需要使用的连接的「激活」按钮
4. 返回「数据管理」页面即可

### Q4: 高级筛选不生效？

确保：
1. 字段名称正确
2. 输入值的类型正确
3. 后端支持对应的筛选参数
4. 点击了「查询」按钮

### Q5: 无法显示中文？

后端需要确保：
1. MySQL 数据库字符集设置为 `utf8mb4`
2. Flask 响应头设置 `Content-Type: application/json;charset=UTF-8`
3. 数据库连接配置中指定字符集

## 📞 技术支持

如有问题，请检查：
1. 浏览器控制台是否有错误信息
2. 网络请求是否正常
3. 后端日志是否有异常
